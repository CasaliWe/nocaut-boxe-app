# 🌐 Nocaut Boxe — Sistema

![Visualização do Site](./nocaut.gif)

## 🧠 Sobre o Projeto

Desenvolvido pela FREELADEV (minha empresa), usando PHP, Bootstrap e SQLite, este sistema web foi criado para otimizar a montagem de treinos personalizados para alunos de musculação. O painel administrativo permite cadastrar grupos musculares e exercícios ilustrados com GIFs, facilitando a criação e organização dos treinos. Cada aluno recebe uma URL exclusiva onde pode acessar seu plano individual e visualizar os exercícios com animações em tempo real — tornando o acompanhamento mais didático e interativo. O projeto foi projetado com foco em simplicidade, performance e experiência do usuário, permitindo que treinadores atualizem treinos de forma ágil e que alunos visualizem suas rotinas de qualquer dispositivo. online disponível em: [https://nocautboxe.com/](https://nocautboxe.com/)

---

## ⚙️ Tecnologias Utilizadas

- PHP puro (back-end)  
- JavaScript para interações front-end  
- Sqlite como banco de dados  
- Bootstrap para estilização  
- Hospedado em servidor Linux (Ubuntu + Nginx)  
- Versionamento com Git + GitHub
